oui.scroll Class

Defined in: src/scroll/scroll.js:2
Module: open-ui

Attach custom scrollbars to an element

HTML:

<div id='scroll'>
                            ...long content....
                        </div>

Javascript:

oui.scroll({
                            element: document.getElementById('scroll'),
                            axis: 'xy'
                        });

Parameters:

  • options Object
    • element Object

      DOM element to attach drag handlers to

    • [sensitivity=60] Number optional

      Number in px to scroll incrementally on mousewheel and key events. Can also be expressed as percentage, e.g. 10% of content

    • [axis=y] Object optional

      Object consisting of x and y {Boolean} values denoting scrollable axis, DOM element to attach drag handlers to. Defaults to element attribute oui-scroll or y
