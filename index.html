Turbo Charge Your UI

OpenUI has been built around the principles of bringing together some of the most common user interface components under a single open source wrapper. Why include multiple libraries, when you can reduce your footprint with a single framework

Dependancy Free, Performant Javascript

With no external dependencies, OpenUI has been expressly written from the ground up to be as light and nimble as possible. All modules and all styling come to a minisule 23kb GZipped! But thats not all, if you arent using certain modules at all whether you include them in the final build is purely optional, reducing size even further.
